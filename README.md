# 🛍️ Product Query Agent with Memory

An AI-powered Product Query Agent that understands natural language product queries and maintains conversation history using **LangGraph Memory**. The agent provides intelligent product recommendations through custom tools and supports multi-turn conversations using the **Groq LLM**.

---

## 🚀 Features

* AI-powered product recommendation agent
* Conversational memory using LangGraph
* Multi-turn interactions
* Custom product lookup tools
* Fast responses with Groq LLM
* Beginner-friendly implementation

---

## 🛠️ Tech Stack

* Python
* LangChain
* LangGraph
* Groq LLM
* Python Dotenv
* Jupyter Notebook

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/codinghub37/product-query-agent-with-memory.git
```

### Navigate to the Project

```bash
cd product-query-agent-with-memory
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file and add your Groq API key:

```env
GROQ_API_KEY=your_groq_api_key
```

---

## ▶️ Run the Project

```bash
jupyter notebook
```

Open **product_query_agent_with_memory.ipynb** and run all notebook cells.

---

## 📚 Workflow

1. Initialize the Groq LLM.
2. Create custom tools for product information.
3. Configure LangGraph Memory to store conversation history.
4. Process user queries using an AI agent.
5. Retrieve relevant product details while remembering previous interactions.
6. Return intelligent, context-aware responses.
